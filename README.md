# Stack & Hustle WordPress Theme

Custom WordPress theme developed for Stack & Hustle, a coworking, tech, and event space in Abuja, Nigeria.

## Live Website
https://stackandhustle.com

## Technologies
- WordPress
- PHP
- CSS
- JavaScript
- BookingPress
- Custom Post Types

## Features
- Workspace booking
- Course listings
- Event pages
- Team profiles
- Gallery pages
- Custom WordPress templates
